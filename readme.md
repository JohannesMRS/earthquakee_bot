# Bot Berita Gempa

Ini merupakan bot telegram yang di buat khusus untuk menampilkan berita gempa terkini dari BMKG

## Cara Menggunakan

### Prasyarat
- **NPM (Node Package Manager)**
- **Node.js**
- **Telegram Account**


### Instalasi
```bash
git clone https://github.com/JohannesMRS/earthquakee_bot.git
cd earthquakee_bot.git
```

### Hal Yang Perlu Dilakukan
- Masuk ke akun telegram
- Cari akun dengan kata kunci 'botfather' (Pilih akun yang memiliki centang)
- Pilih dan ketik opsi /newbot
- Nanti akan diminta untuk membuat nama & username dari bot nya
- Jika username available maka akan muncul informasi Token acces HTTP API
- Replace value dari variable token di reository ini dengan token milik kamu
- Untuk testing bot, di pesan yang sama untuk token, klik link t.me/'nama_bot_kamu'

### Cara Menggunakan
```bash
npm run bot
```
